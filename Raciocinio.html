Raciocínio Algorítmico

Resolução de Problemas(1)

O primeiro projeto da disciplina de Raciocínio Algorítmico, você e sua equipe foram incumbidos de resolver problemas usando a linguagem Python em sua versão 3. Cada enunciado abaixo deverá, obrigatoriamente, ser resolvido em um arquivo Python (.py).

Sistema bancário(2)

O segundo projeto da disciplina de Raciocínio Algorítmico, você e sua equipe foram contratados para realizar a implementação de um sistema bancário usando a linguagem Python versão 3. O sistema deverá ser executado via linha de comando e os dados deverão ser armazenados toda vez que o sistema for fechado. Para armazenar os dados, sugere-se utilizar o pacote pickle.

O sistema deverá possuir duas interfaces, sendo que cada uma deverá ser implementada em um arquivo Python diferente: (1.) Interface de gerente (2.) Interface de cliente.

Simulação de Alocação de Memória (3)

O terceiro projeto da disciplina de Raciocínio Algorítmico, você e sua equipe foram incumbidos de desenvolver um simulador de memória. A memória deverá ser simulada como uma matriz, onde cada posição determina se o espaço em questão está ocupado ou não, isto é, usando valores booleanos.
