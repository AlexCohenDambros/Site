Experiência Criativa: Navegando na Computação

Projeto do Jogo (1)

O primeiro projeto da disciplina de Experiência Criativa é o desenvolvimento de um JOGO utilizando a ferramenta CONSTRUCT. O grupo foi formado por: João Guilherme Gusso, Alex Cohen Dambrós Lopes, Anderson e Eduardo Nakatani.

O objetivo do Trabalho: Verificar as características e limitações da ferramenta, criar o conceito do jogo que se encaixe nestas características e limitações. Estudar a ferramenta nas modalidades de desenvolvimento. A partir do conceito do jogo detalhar o suficiente para iniciar o processo de desenvolvimento. Depois de uma primeira versão, detalhar mais e desenvolver a segunda versão - e assim por diante.

Projeto do Aplicativo Multimídia (2)

O segundo projeto da disciplina de Experiência Criativa é o desenvolvimento de um Aplicativo Multimídia, usando a ferramenta Processing (Atividade Individual). Especificação do Projeto: aplicativo desenvolvido no Processing que contenha os diversos tipos de mídia (som, fotografia, desenho, vídeo), seja interativo e tenha um propósito específico. Como por exemplo, jogo ou objeto educacional.

O objetivo do Trabalho: além do estudo da ferramenta (IDE) e suas características e funcionalidades, deverão ser nas diversas formas de se armazenar e manipular os diferentes tipos de informação (texto, imagem, áudio, vídeo) e alternativas de interação.

Projeto Ferramenta Profissional Visual Studio (3)

O terceiro projeto da disciplina de Experiência Criativa é o desenvolvimento de um Aplicativo em C#, usando a ferramenta Visual Studio e usando o recurso Windows Forms (Atividade Individual).

O objetivo do Trabalho: Ferramenta (IDE) MS Visual Studio (interface, funcionalidades, usabilidade, estabilidade, performance, suporte a múltiplas linguagens e recursos para automação da programação), Linguagem C# (origem, semelhança com outras linguagens, facilidade de aprendizado e uso), Biblioteca para o desenvolvimento de interface gráfica (GUI) Windows Forms (facilidade de uso, eficiência, recursos, suporte a linguagens) e Processo de desenvolvimento de software com os três itens citados acima.
